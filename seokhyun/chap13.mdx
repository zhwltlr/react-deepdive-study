## 웹페이지의 성능을 측정하는 다양한 방법

### 웹페이지의 성능 측정 방법

- 웹페이지의 성능을 측정하는 방법은 다양합니다.
- 대표적인 성능 측정 도구로는 크롬 개발자 도구, Lighthouse, PageSpeed Insights 등이 있습니다.
- reportWebvitals 함수를 사용하면 create-react-app 프로젝트에서 웹 페이지의 성능을 측정할 수 있습니다.
- reportWebvitals 함수는 Core Web Vitals 지표인 LCP, FID, CLS를 측정하여 성능을 평가합니다.
- NextWebVitals 컴포넌트를 사용하면 Next.js 프로젝트에서 웹 페이지의 성능을 측정할 수 있습니다.
